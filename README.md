# hello-react-front-end

A react-rails app to show greetings

## Built With

- React
- Redux

## Getting Started

To get a local copy up and running follow these simple example steps.

## Prerequisites
node, npm

## Setup
clone repo
cd repo
Install
npm install

## Usage
browse localhost://3000
